演算ボタン

使用方法

2つの数の四則演算が可能です。
「＝」はなく、「＋」「－」「×」「÷」を選択していると、リアルタイムで計算結果が反映されます。

ボタンショートカット用途
0 ～ 90 ～ 9(10キーも可)数の入力
00なし
.D または 10キーの「.」小数点の入力
ACAオールクリア
ClearC選択している数のクリア
TabTabキー または T入力場所の切替
±P または M選択している数の正負の変更
＋ － × ÷10キーの+ - * /四則演算の切替
