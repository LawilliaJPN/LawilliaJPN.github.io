Web制作の練習 #03 Counter

COUNTER
NAME:

使用方法

「NAME:」に名前を入力し、「ADD」でカウンタを作成します。
「NAME:」が空欄の場合は「Counter1」「Counter2」……という名前になります。
「NAME:」に名前を入力し、「DELETE」でカウンタを削除します。

ボタンショートカット用途
DELETEなし入力した名前のカウンタの削除
CLEARなしすべてのカウンタの削除
ADDなし入力した名前のカウンタの新規作成
＋なし各カウンタの数値の増減
－なし
